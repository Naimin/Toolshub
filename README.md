# Toolshub

A collection of useful web-based tools and calculators.

## Tools

### Compound Interest Calculator
A beautiful and interactive compound interest calculator that helps you visualize your investment growth over time.

**Features:**
- Calculate compound interest with customizable parameters
- Visualize growth with interactive charts
- Support for different compounding frequencies
- Detailed breakdown of principal and interest

## Getting Started

Simply open the HTML files in your web browser. No installation or build process required.
